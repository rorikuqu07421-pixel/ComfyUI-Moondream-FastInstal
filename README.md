### 🔥 One-Click Install (Recommended)

1. Open **ComfyUI**
2. Click the **Manager** button
3. Go to **Install Custom Nodes**
4. Search: `moondream` or `ComfyUI-Moondream-Easy`
5. Click **Install** → Restart ComfyUI

**✅ Moondream node is ready!**
